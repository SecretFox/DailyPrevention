# DailyPrevention  
Prevents your other characters from stealing daily login rewards, event rewards can still be claimed as usual.  
To override the mod you can hold Ctrl while clicking the Claim button.  
To configure which characters are allowed to claim login rewards run `/option DailyPrevention_Character "<CharacterName>"` chat command, you can also separate multiple characters with commas.  

** AutoClaim **  
To enable auto claiming of login rewards run `/option DailyPrevention_Autoclaim true` chat command.  
Autoclaim will claim event rewards on all characters, but will not touch the normal login rewards if the character is not whitelisted.  

**Installation**  
Extract to ..\Secret World Legends\Data\Gui\Custom\Flash\ folder.
