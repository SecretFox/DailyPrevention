# DailyPrevention
Prevents your alts from stealing your daily  cache keys, event rewards should still be claimable on all characters.  
If you want to claim daily reward on your alt you have to be holding ctrl while clicking the claim button.  
Install as normal and and run `/option DailyPrevention_Character "<CharacterName>"` command, you can also separate multiple characters with comma.  

**Installation**  
Unzip to ..\Secret World Legends\Data\Gui\Custom\Flash\ folder.
