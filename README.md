# DailyPrevention
Prevents your alts from stealing your daily  cache keys.  
If you want to claim daily reward on your alt you have to be holding ctrl while clicking the claim button.  
Events reward claiming probably also requires you to hold Ctrl.  
Install as normal and fill your character name into LoginPrefs.xml file, you can use comma separated list too.  

**Installation**  
Unzip to ..\Secret World Legends\Data\Gui\Custom\Flash\ folder.
